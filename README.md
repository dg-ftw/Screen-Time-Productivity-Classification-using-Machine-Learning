# Screen-Time-Productivity-Classification-using-Machine-Learning

This repository contains a machine learning project for analyzing screen time, sleep, exercise, and productivity patterns.

## Files

- `screen_time_productivity_classifier.py` - main analysis and model training script
- `cleaned_form.csv` - survey dataset
- `kaggle_screentime.csv` - Kaggle screen time dataset
- `model_output.png` - generated model comparison output

## Run

Install the Python dependencies, then run:

```powershell
python screen_time_productivity_classifier.py
```
