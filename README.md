# ML Project

This repository contains a machine learning project for analyzing screen time, sleep, exercise, and productivity patterns.

## Files

- `python project (1).py` - main analysis and model training script
- `cleaned_form.csv` - survey dataset
- `kaggle_screentime.csv` - Kaggle screen time dataset

## Run

Install the Python dependencies, then run:

```powershell
python "python project (1).py"
```
